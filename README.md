## 📖 About

This is a fully responsive, professional college website built with pure **HTML, CSS, and JavaScript** — no frameworks required. It includes a built-in **Admin Panel** that allows authorized users to edit any section of the website directly from the browser, without touching any code.

---

## ✨ Features

- ✅ **Fully Responsive** — Mobile, Tablet & Desktop ready
- ✅ **Built-in Admin Panel** — Edit any section live in the browser
- ✅ **Click-to-Edit** — Click any element to open its HTML/CSS editor
- ✅ **localStorage Saving** — Changes persist after page reload
- ✅ **Global CSS Editor** — Edit website styles from admin panel
- ✅ **Delete Elements** — Remove any section with one click
- ✅ **Smooth Animations** — Scroll reveal, hover effects, transitions
- ✅ **Sticky Header** — With dropdown navigation menus
- ✅ **Marquee Notice Bar** — For announcements & admissions info
- ✅ **Contact Form** — With animated submit button
- ✅ **No Dependencies** — Pure HTML/CSS/JS, no npm or build tools needed

## 🔐 Admin Panel

This website includes a **fully functional Admin Panel** built into the same HTML file.

### How to Access

| Method | Action |
|---|---|
| ⚙️ Gear Button | Click the **gear icon** at bottom-right of the page |
| ⌨️ Keyboard Shortcut | Press `Ctrl + Shift + A` |

### Default Credentials

```
Username : admin
Password : admin123
```

> ⚠️ **Important:** Change the password before deploying to production.  
> Open the file, find `PASSWORD: 'admin123'` in the JavaScript and replace it.

### Admin Features

- **✏️ Edit Mode** — Hover over any element to highlight it, click to open editor
- **📄 HTML Editor** — Edit the full HTML of any clicked element
- **🎨 CSS Editor** — Edit global CSS styles from one place
- **✓ Apply** — Apply changes instantly and see them live
- **🗑 Delete** — Remove any element from the page
- **💾 Save All** — Save all changes to localStorage
- **🔄 Reset** — Restore original website
- **📋 Copy** — Copy code to clipboard
- **⎇ Format** — Auto-format HTML code

## 🔧 How to Use Admin Panel

```
1. Open the website in browser
2. Click ⚙️ gear button (bottom-right) OR press Ctrl+Shift+A
3. Enter password: admin123
4. Click "Edit Mode: OFF" to turn Edit Mode ON
5. Hover over any element → blue dotted outline appears
6. Click the element → Editor panel slides from right
7. Edit HTML in the editor
8. Click "Apply HTML" to see changes live
9. Click "Save All" to save changes permanently
10. To reset everything → click "Reset to Default"
```
